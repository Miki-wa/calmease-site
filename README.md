# calmease-site
ハンドメイド作品の魅力を伝えるための紹介サイトです。

## URL
https://miki-wa.github.io/calmease-site/

## 概要
友人が制作したぬいぐるみ作品の魅力を、写真だけでは伝えきれないと感じ、  
世界観やコンセプトを丁寧に表現することを目的として制作しました。

## 目的
作品の世界観やコンセプトを伝え、ECサイトやイベントへの導線を作ること

## ターゲット
ハンドメイド作品やぬいぐるみに興味があり、  
日常の中で癒しやこだわりを大切にする20代以上の層

## 使用技術
- HTML
- CSS（レスポンシブ対応）

## 工夫した点
- 「落ち着き」や「安心感」を表現するため、配色と余白を意識したデザイン
- 作品を生活に取り入れたイメージが伝わるよう、視覚的に分かりやすい構成
- スマートフォンでの閲覧を意識した視認性・操作性への配慮

## 改善案
- 作品数増加に対応するためのカテゴリ分けや検索機能の追加
- ECサイトへの導線を明確にし、購入への遷移率向上を図る
